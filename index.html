Разверните образовательную платформу за 1 минуту

Техническая часть универсальна. Далее вы наполняете её своим курсом, правилами форума и фирменным стилем.

Создайте аккаунт на GitHub

Бесплатная регистрация на github.com (2 минуты)

Откройте базовый шаблон

Перейдите по ссылке: github.com/feekool/feekool.github.io

Скопируйте в свой репозиторий

Нажмите Use this template → Create a new repository. Назовите, например, nutrilearn-course

Включите публикацию (GitHub Pages)

В настройках: Settings → Pages → ветка main → Save. Сайт появится через ~60 сек по адресу никнейм.github.io/репозиторий

Настройте под обучение

Замените заголовки, добавьте разделы «Модули», «Домашки», «Вопросы куратору». Загрузите PDF, добавьте правила общения на форуме.

💡 Совет: Запустите MVP: 2 модуля + 1 ветка форума «Быстрый старт». По мере набора студентов добавляйте «Проверка заданий» и «Закрытый клуб выпускников».
